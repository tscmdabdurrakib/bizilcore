ONBOARDING OPTIMIZATION ===

━━━ SETUP CHECKLIST (Dashboard — new users only) ━━━

Show checklist card on dashboard until:
  All tasks completed OR user dismisses after 70%+ complete

Database addition to User:
  setupProgress Json? -- stores which tasks are done

Checklist tasks (auto-checked when action is done):
  [✓] Account তৈরি করেছেন           +10 XP  (always checked)
  [ ] Shop profile সম্পূর্ণ করুন    +20 XP  → /settings
  [ ] প্রথম পণ্য যোগ করুন            +30 XP  → /inventory/new
  [ ] প্রথম customer যোগ করুন        +20 XP  → /customers/new
  [ ] প্রথম অর্ডার তৈরি করুন         +50 XP  → /orders/new
  [ ] প্রথম হিসাব এন্ট্রি করুন       +20 XP  → /hisab

Checklist card UI:
  Header: "Setup সম্পন্ন করুন" + progress percentage (e.g. "৪০%")
  Progress bar (green fill, animated)
  Each task row:
    Checkbox (checked = green filled) | Task text | XP badge | "করুন →" button
    Completed tasks: strikethrough + faded
  
  At 100%:
    Confetti animation
    Card transforms to: "🎉 Setup সম্পন্ন! আপনি এখন প্রস্তুত।"
    Auto-hides after 5 seconds
  
  "এখন না" button: hides card for 24 hours (not permanent dismiss)
  After 70% complete: "Dismiss" button appears (permanent hide)

Auto-check tasks:
  After POST /api/shops/update (with all required fields) → mark "profile_complete"
  After POST /api/products (first product) → mark "first_product"
  After POST /api/customers (first customer) → mark "first_customer"
  After POST /api/orders (first order) → mark "first_order"
  After POST /api/transactions (first transaction) → mark "first_transaction"
