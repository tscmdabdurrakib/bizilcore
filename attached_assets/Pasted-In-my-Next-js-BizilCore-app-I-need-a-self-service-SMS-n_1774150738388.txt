### 3. API Routes

**POST `/api/settings/sms/connect`**
- Accept { api_key }
- Verify API key by calling SMS.net.bd balance check endpoint:
  GET https://api.sms.net.bd/balance?api_key={api_key}
- If valid (returns balance): save encrypted api_key, set is_connected = true
  Also show remaining SMS balance to user
- If invalid: return error "API Key সঠিক নয়"

**POST `/api/settings/sms/disconnect`**
- Set is_connected = false, delete api_key for current user

**GET `/api/settings/sms`**
- Return current user's sms settings and notification preferences
- Return masked api_key: "••••••••xxxx"
- Return current SMS balance if connected

**PUT `/api/settings/sms/preferences`**
- Update sms_notification_preferences table for current user

**POST `/api/settings/sms/test`**
- Accept { phone }
- Send test SMS using user's own api_key
- Message: "BizilCore SMS সংযোগ সফল! আপনার নোটিফিকেশন সিস্টেম চালু আছে।"
- Use SMS.net.bd API:
  POST https://api.sms.net.bd/sendsms
  Body: { api_key, msg, to }

### 4. lib/sms.js utility

export async function sendSMS(apiKey, toPhone, message) {
  // Auto-format: 01XXXXXXXXX stays as is for SMS.net.bd
  const response = await fetch("https://api.sms.net.bd/sendsms", {
    method: "POST",
    headers: { "Content-Type": "application/json" },
    body: JSON.stringify({
      api_key: apiKey,
      msg: message,
      to: toPhone,
    }),
  });
  return response.json();
}

export async function checkSMSBalance(apiKey) {
  const response = await fetch(
    `https://api.sms.net.bd/balance?api_key=${apiKey}`
  );
  return response.json();
}

export async function shouldSendSMS(userId, preferenceKey) {
  const settings = await getUserSMSSettings(userId);
  if (!settings?.is_connected) return false;
  return settings?.preferences?.[preferenceKey] === true;
}

### 5. Trigger SMS on Order Events

After order is created successfully:

const smsEnabled = await shouldSendSMS(userId, "order_confirmed");
if (smsEnabled) {
  const settings = await getUserSMSSettings(userId);
  const customerPhone = order.customer.phone;
  const message = 
    `আপনার অর্ডার #${order.id} কনফার্ম হয়েছে! ` +
    `মোট: ${order.total} টাকা। ` +
    `ধন্যবাদ।`;

  await sendSMS(settings.api_key, customerPhone, message);
}
