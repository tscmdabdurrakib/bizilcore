### 4. API Routes

**POST /api/settings/whatsapp/connect**
- Accept { api_token, phone_number_id, business_account_id }
- Verify by calling Meta API:
  GET https://graph.facebook.com/v18.0/{phone_number_id}
  Headers: Authorization: Bearer {api_token}
- If valid: encrypt and save credentials, set is_connected = true
- If invalid: return 400 with error message

**POST /api/settings/whatsapp/disconnect**
- Clear credentials, set is_connected = false

**GET /api/settings/whatsapp**
- Return connection status
- Return masked token (show last 6 chars only: "••••••••abc123")
- Never return raw token

**POST /api/settings/whatsapp/test**
- Accept { phone }
- Send test message using user's credentials
- Message: "✅ আপনার BizilCore WhatsApp সংযোগ সফল হয়েছে! 🎉"

**POST /api/communications/whatsapp/send**
- Accept { to_phone, message, customer_id (optional) }
- Get user's decrypted api_token and phone_number_id from DB
- Auto-format phone: 01XXXXXXXXX → 880XXXXXXXXXX
- Send via Meta API:
  POST https://graph.facebook.com/v18.0/{phone_number_id}/messages
  {
    messaging_product: "whatsapp",
    to: formatted_phone,
    type: "text",
    text: { body: message }
  }
- Save to message_logs table:
  { user_id, customer_id, to_phone, message, status, sent_at }
- Return success or error

**GET /api/communications/whatsapp/history**
- Return paginated message history for current user
- Include customer name if customer_id exists

### 5. lib/whatsapp.js

export async function sendWhatsAppMessage(apiToken, phoneNumberId, toPhone, message) {
  const formattedPhone = toPhone.startsWith("0") 
    ? "88" + toPhone 
    : toPhone;

  const res = await fetch(
    `https://graph.facebook.com/v18.0/${phoneNumberId}/messages`,
    {
      method: "POST",
      headers: {
        "Authorization": `Bearer ${apiToken}`,
        "Content-Type": "application/json",
      },
      body: JSON.stringify({
        messaging_product: "whatsapp",
        to: formattedPhone,
        type: "text",
        text: { body: message },
      }),
    }
  );

  const data = await res.json();
  if (data.error) throw new Error(data.error.message);
  return data;
}

export async function verifyWhatsAppCredentials(apiToken, phoneNumberId) {
  const res = await fetch(
    `https://graph.facebook.com/v18.0/${phoneNumberId}`,
    {
      headers: { "Authorization": `Bearer ${apiToken}` },
    }
  );
  return res.ok;
}
