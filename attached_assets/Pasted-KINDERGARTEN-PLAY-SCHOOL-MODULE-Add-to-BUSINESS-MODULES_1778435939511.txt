━━━ DATABASE ━━━

(Reuse existing Student, Batch, FeeRecord, AttendanceRecord from School module)
(Add kindergarten-specific models)

Add to Student model (kindergarten extras):
  section        String?   -- "Playgroup"|"Nursery"|"KG-1"|"KG-2"|"Pre-School"
  foodAllergies  String?   -- CRITICAL: "Peanut allergy", "Lactose intolerant"
  pickupPerson1  String?   -- "Father - Karim" (who can pick up)
  pickupPhone1   String?
  pickupPerson2  String?   -- backup person
  pickupPhone2   String?
  medicalNote    String?   -- doctor contact, conditions

model MealRecord {
  id        String   @id @default(cuid())
  shopId    String
  studentId String
  student   Student  @relation(fields: [studentId], references: [id])
  date      DateTime
  mealType  String   -- "breakfast"|"lunch"|"snack"|"water"
  status    String   -- "ate_all"|"ate_half"|"didnt_eat"|"brought_from_home"
  note      String?
  @@unique([studentId, date, mealType])
}

model DailyReport {
  id           String   @id @default(cuid())
  shopId       String
  studentId    String
  student      Student  @relation(fields: [studentId], references: [id])
  reportDate   DateTime
  mood         String?  -- "happy"|"sad"|"cranky"|"tired"|"energetic"
  activities   String?  -- "drawing, singing, outdoor play"
  ate          String?  -- summary of meals
  napped       Boolean  @default(false)
  napDuration  Int?     -- minutes
  teacherNote  String?
  sentToParent Boolean  @default(false)
  sentAt       DateTime?
  @@unique([studentId, reportDate])
}

━━━ KINDERGARTEN DASHBOARD (/dashboard) ━━━

4 stat cards:
  মোট শিশু: active children count
  আজকের উপস্থিতি: present today count
  Food Allergy সতর্কতা: count of children with food allergies
  এই মাসের আয়: fee revenue this month

Today's allergy alert banner (if any child with allergy is present today):
  Red banner: "⚠️ আজ উপস্থিত শিশুদের মধ্যে food allergy আছে:"
  List: child name + allergy type
  Always visible, cannot dismiss (safety critical)

Today's attendance quick summary:
  উপস্থিত: X | অনুপস্থিত: Y | Total: Z

Daily report status:
  "আজকের [X/Y] শিশুর report পাঠানো হয়েছে"
  "বাকি [Z] জনের report পাঠান" button

    Name*, date of birth*, gender*

  অভিভাবকের তথ্য:
