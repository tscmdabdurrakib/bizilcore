=== FEATURE: BIZILCORE STORE — MULTI-TENANT E-COMMERCE ===
(BizilCore SaaS এর ভেতরে একটি নতুন feature)

━━━ CONCEPT ━━━
প্রতিটি BizilCore shop owner পাবে তাদের নিজস্ব 
e-commerce website:
  URL: bizilcore.com/store/[shopSlug]
  অথবা custom domain: www.theirshop.com

এটা Next.js App Router দিয়ে বানানো SPA।
কোনো page reload হবে না — সব client-side navigation।
BizilCore dashboard এর data automatically sync হবে।

━━━ DATABASE ADDITIONS ━━━

Add to Shop model:
  storeSlug         String?   @unique
  storeEnabled      Boolean   @default(false)
  storeTheme        String    @default("default")
  storePrimaryColor String?
  storeAccentColor  String?
  storeBannerUrl    String?
  storeTagline      String?
  storeAbout        String?
  storeShowReviews  Boolean   @default(true)
  storeShowStock    Boolean   @default(false)
  storeCODEnabled   Boolean   @default(true)
  storeBkashNumber  String?
  storeNagadNumber  String?
  storeMinOrder     Float?
  storeFreeShipping Float?
  storeShippingFee  Float     @default(60)
  storeCustomDomain String?
  storeSocialFB     String?
  storeSocialIG     String?
  storeSocialWA     String?

model StoreOrder {
  id              String    @id @default(cuid())
  shopId          String
  shop            Shop      @relation(fields: [shopId], references: [id])
  orderNumber     String    @unique
  customerName    String
  customerPhone   String
  customerAddress String
  customerDistrict String?
  customerNote    String?
  items           StoreOrderItem[]
  subtotal        Float
  shippingFee     Float     @default(60)
  discountAmount  Float     @default(0)
  totalAmount     Float
  paymentMethod   String    -- "cod"|"bkash"|"nagad"
  paymentStatus   String    @default("pending")
  status          String    @default("pending")
  couponCode      String?
  ipAddress       String?
  createdAt       DateTime  @default(now())
}

model StoreOrderItem {
  id          String     @id @default(cuid())
  storeOrderId String
  storeOrder  StoreOrder @relation(fields: [storeOrderId], references: [id])
  productId   String
  product     Product    @relation(fields: [productId], references: [id])
  variantId   String?
  productName String
  quantity    Int
  unitPrice   Float
  subtotal    Float
}

model ProductVariant {
  id        String  @id @default(cuid())
  productId String
  product   Product @relation(fields: [productId], references: [id])
  name      String  -- "Size: L", "Color: Red"
  type      String  -- "size"|"color"|"material"
  value     String
  priceAdj  Float   @default(0)
  stockQty  Int     @default(0)
  imageUrl  String?
}

model StoreReview {
  id         String   @id @default(cuid())
  shopId     String
  productId  String?
  product    Product? @relation(fields: [productId], references: [id])
  orderId    String?
  reviewerName String
  rating     Int      -- 1-5
  comment    String?
  isApproved Boolean  @default(false)
  createdAt  DateTime @default(now())
}

model Coupon {
  id          String   @id @default(cuid())
  shopId      String
  code        String
  type        String   -- "percent"|"fixed"
  value       Float
  minOrder    Float?
  maxDiscount Float?
  usedCount   Int      @default(0)
  maxUse      Int?
  expiresAt   DateTime?
  isActive    Boolean  @default(true)
  @@unique([shopId, code])
}

━━━ THEMES — ৫টি বানাতে হবে ━━━

Create /themes folder with 5 theme configs:

1. "elegant" — Premium fashion theme
   Colors: Black bg, gold accent, white text
   Layout: Full-width hero, grid products, minimal nav

2. "vibrant" — Colorful lifestyle theme
   Colors: White bg, bright pink/purple accent
   Layout: Banner slider, category icons, card products

3. "minimal" — Clean modern theme
   Colors: White bg, dark text, single accent color
   Layout: Simple nav, list+grid toggle, clean cards

4. "bold" — Strong brand theme
   Colors: Dark bg, bright green (like BizilCore)
   Layout: Big typography, featured product hero

5. "classic" — Traditional shop theme
   Colors: Light gray bg, blue accent
   Layout: Traditional e-commerce layout like Daraz

Each theme is a ThemeConfig object:
{
  id: "elegant",
  name: "Elegant",
  preview: "/themes/elegant-preview.jpg",
  config: {
    navStyle: "centered"|"left"|"minimal",
    heroStyle: "fullwidth"|"split"|"banner",
    productCard: "square"|"tall"|"horizontal",
    fontHeading: "Playfair Display"|"Inter"|"Hind Siliguri",
    fontBody: "Inter"|"Hind Siliguri",
    borderRadius: "none"|"sm"|"lg"|"full",
    shadows: boolean,
    animations: boolean
  }
}

━━━ STORE PAGES (all under /store/[slug]/) ━━━

All pages are Next.js App Router with:
  - No full page reload (use next/link everywhere)
  - Framer Motion page transitions
  - Skeleton loading states
  - Mobile-first responsive

1. HOME PAGE (/store/[slug])

Sections (order based on theme):
  a. NAVBAR:
     - Shop logo + name
     - Search bar (searches products instantly)
     - Cart icon with item count badge
     - "অর্ডার ট্র্যাক করুন" link
     
  b. HERO BANNER:
     - Shop banner image (uploaded by owner)
     - Shop tagline text
     - "এখনই কিনুন" CTA button

     - Shop about text

2. PRODUCT LIST PAGE (/store/[slug]/products)

3. PRODUCT DETAIL PAGE (/store/[slug]/products/[productId])

import { persist } from 'zustand/middleware'

const useCart = create(persist((set, get) => ({
  items: [],
  shopSlug: null,
  addItem: (product, variant, qty) => ...
  removeItem: (productId, variantId) => ...
  updateQty: (productId, variantId, qty) => ...
  clearCart: () => ...
  total: () => items.reduce(...)
}), { name: 'bizilcore-cart' }))

Cart clears automatically when shop changes
(prevent mixing items from different shops)

━━━ THEME SYSTEM ━━━

ThemeProvider wraps entire /store/[slug] layout:

// layout.tsx
const shop = await getShop(slug)
const theme = themes[shop.storeTheme]

return (
  <ThemeProvider theme={theme} colors={{
    primary: shop.storePrimaryColor || theme.defaults.primary,
    accent: shop.storeAccentColor || theme.defaults.accent,
  }}>
    <StoreNavbar shop={shop} />
    {children}
    <StoreFooter shop={shop} />
  </ThemeProvider>
)

ThemeProvider injects CSS variables:
:root {
  --store-primary: [primaryColor];
  --store-accent: [accentColor];
  --store-radius: [borderRadius from theme];
  --store-font-heading: [fontHeading from theme];
}

━━━ SEO ━━━

Each store page has proper metadata:

// /store/[slug]/page.tsx
export async function generateMetadata({ params }) {
  const shop = await getShop(params.slug)
  return {
    title: `${shop.name} — অনলাইন শপ`,
    description: shop.storeTagline || 
      `${shop.name} এর অনলাইন শপে স্বাগতম`,
    openGraph: {
      images: [shop.storeBannerUrl || shop.logoUrl]
    }
  }
}
