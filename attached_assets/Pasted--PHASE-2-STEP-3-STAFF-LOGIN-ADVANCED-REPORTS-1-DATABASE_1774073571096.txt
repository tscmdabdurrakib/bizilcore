--- 1. DATABASE UPDATE ---
Add StaffMember model:
model StaffMember {
  id         String   @id @default(cuid())
  shopId     String
  shop       Shop     @relation(fields: [shopId], references: [id])
  userId     String
  user       User     @relation(fields: [userId], references: [id])
  role       String   -- "manager" or "staff"
  inviteToken String? @unique
  invitedAt  DateTime @default(now())
  joinedAt   DateTime?
  isActive   Boolean  @default(true)
}

Add ActivityLog model:
model ActivityLog {
  id        String   @id @default(cuid())
  shopId    String
  userId    String
  action    String   -- "order_created", "product_edited", etc.
  detail    String?
  createdAt DateTime @default(now())
}

- "Staff invite করুন" button → modal:

  - "Invite পাঠান" → generates invite token → sends SMS or email

Create /invite/[token] page:
- Shows shop name + inviter name
- "Accept করুন" → creates user account (if new) → links to shop as staff
- Role-based middleware: check staffMember.role before allowing actions

Staff permission matrix:
  Manager: can do everything except settings and billing
  Staff: can only view + create orders, cannot delete or edit prices
